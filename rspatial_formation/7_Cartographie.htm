7. Ralisation de cartes thmatiques avec R

De nombreuses possibilits

Jusqu maintenant nous avons vu comment visualiser rapidement les rsultats de nos analyses spatiales avec ggplot() et le plot() de base.

De nombreux packages R permettent de raliser de vritables cartes avec lgendes, chelle, nord etc. Certains packages sont des  add-on  de ggplot comme ggmap ou ggspatial, dautres sont des packages  indpendants  permettant la cration de cartes thmatiques bases sur des donnes vectorielles et raster spatialises comme tmap, mapsf, cartography etc. Dautres sont des packages  spcialiss  comme rayshader pour des cartes 3D du terrain, leaflet ou mapdeck (3D) pour des cartes web. Pour crer des cartes animes, le package gganimate peut tre utilis.

Dans la prsentation 1_Introduction, vous trouverez une galerie de ce qui est possible avec ces diffrents packages. Les connaissances acquises jusque maintenant devraient vous permettre de comprendre (donc modifier pour votre usage) la plupart des codes-sources prsents dans les notes de bas de page de cette prsentation.

# Probleme si beacoup de subplot : de nombreux copier-coller

# Solution : faire une fonction qui prend en arguments la colonne et la palette souhaite

 

make_subcat_plot <- function(param, palette_g, direction_g){

subcat_plot <- ggplot() +

geom_sf(data = subcat_sf,

aes(fill = .data[[param]]),

color = NA) +

scale_fill_distiller(palette = palette_g, direction = direction_g) +

scale_x_continuous(breaks = seq(-2.4,-2.2,0.1)) +

scale_y_continuous(breaks = seq(48,48.2,0.1)) +

labs(title = param) +

theme_bw(base_size = 8) +

theme(legend.position = "bottom")

return(subcat_plot)

}

make_subcat_plot("URBAIN", "Greys", 1) + make_subcat_plot("FORET", "Greens", 1)
